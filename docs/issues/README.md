# Issue Documentation

This directory contains documentation for issues, epics, and user stories related to the PinyinPal application.

## Directory Structure

```
/docs/issues/
  ├── README.md                       # This file
  ├── epic-1-google-cloud-tts-integration.md  # Epic 1: TTS Integration
  ├── epic-2-vocabulary-learning-flow.md      # Epic 2: Vocabulary Learning Flow
  ├── 007-google-cloud-tts-integration-issue.md  # GitHub issue for Epic 1
  └── epic-2-stories/                 # User stories for Epic 2
      ├── story-1-select-vocabulary-list.md
      ├── story-2-set-daily-commitment.md
      ├── story-3-divide-list-into-sections.md
      ├── story-4-select-section-for-learning.md
      ├── story-5-update-flashcard-page.md
      ├── story-6-manage-tracking-data.md
      └── story-7-export-import-tracking-data.md
```

## Naming Convention

- **Epic files**: `epic-[number]-[descriptive-name].md`
- **GitHub issue files**: `[issue-number]-[descriptive-name]-issue.md`
- **User story files**: `story-[number]-[descriptive-name].md`

## File Structure

### Epic Documents

Epic documents contain:

- Overview of the epic
- Background information
- Implementation details
- List of user stories
- Status

### User Story Documents

User story documents contain:

- Description
- Acceptance criteria
- Implementation notes
- Status

### GitHub Issue Files

GitHub issue files contain:

- Title
- Description
- Goals
- Technical requirements
- Acceptance criteria
- Implementation details
- Priority
- Labels
